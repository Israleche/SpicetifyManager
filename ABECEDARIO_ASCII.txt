================================================================================
        ANSI SHADOW STYLE - 6-LINE ASCII ALPHABET  (Reference Sheet)
================================================================================

Description
-----------
Each glyph is drawn on a 6-line grid using block characters (█ ╗ ╔ ╝ ╚ ║ ═)
that render correctly on any UTF-8 terminal with a monospaced font. All glyphs
in this sheet occupy the same height, so they stack into clean banners when
concatenated horizontally.

How to use
----------
1. Pick the glyphs you need (one letter block per character of your title).
2. Concatenate the same row index across all glyphs to build each banner line.
3. The result is a 6-line tall banner. Pad with spaces between glyphs for
   readability.

Width notes
-----------
- Most letters are 6 columns wide.
- Some letters (I, J, 1, !, ., :, ;) are narrower (2-3 columns).
- M and W are 8 columns wide.
- Tweak per-glyph spacing when mixing narrow and wide letters.
